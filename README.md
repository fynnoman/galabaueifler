# GALA BAUER - Garten & Landschaftsbau Website

A modern, clean website for a German landscaping company built with Next.js, React, and Tailwind CSS.

## Design Philosophy

- Clean, minimal design inspired by German industrial aesthetics
- Big, bold headings with minimal text
- No shadows, gradients, or light effects
- Limited color palette - primarily neutral colors
- Mobile-responsive design

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/src/app` - Next.js app directory with pages and layouts
- `/src/components` - React components (Header, Hero, Services, About, Gallery, Contact, Footer)
- `/public/images` - Image assets (placeholder images included)

## Customization

### Replace Placeholder Images

Replace the placeholder images in `/public/images/` with actual photos of:
- Landscaping projects
- Garden designs
- Team at work
- Completed projects

### Update Contact Information

Edit `/src/components/Contact.tsx` to update:
- Company name and address
- Phone number
- Email address
- Opening hours

### Modify Services

Edit `/src/components/Services.tsx` to add, remove, or modify the services offered.

## Tech Stack

- **Next.js 16** - React framework with App Router
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **React** - Component-based UI library

## Build for Production

```bash
npm run build
npm start
```
